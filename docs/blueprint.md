# **App Name**: PDF Scry

## Core Features:

- PDF Upload: Allow users to upload PDF files from their local storage.
- Content Extraction: Parse the uploaded PDF to identify text, tables, and images.
- Content Display: Display the extracted content, categorizing it into text, tables, paragraphs, and images.
- Text Download: Enable users to download all extracted text as a single plain text file.
- Image Download: Allow users to download individual or multiple extracted images.
- AI-Powered Summarization: Incorporate an AI tool that analyzes the extracted text and identifies key topics, creating a summary or outline of the PDF's content.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to convey professionalism and stability.
- Background color: Light grey (#F2F2F2) for a clean and unobtrusive backdrop.
- Accent color: Light blue (#A9BCD0) to highlight interactive elements and important information.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its clean and modern appearance.
- Use a set of minimalistic line icons to represent different content types (text, image, table).
- Employ a well-structured layout with clear sections for uploaded file display, extracted text, tables, and images. Utilize grid or flexbox layouts for responsiveness.
